## Pronouns & Determiners

| Patois | English | Example | Notes |
|--------|---------|---------|-------|
| mi | I / me / my | "Mi a work pon it" (I'm working on it) | Replaces 3 English words with 1 |
| yuh | you / your | "Yuh code clean" (Your code is clean) | |
| im | he / she / him / her / his / its | "Im nuh work" (It doesn't work) | Gender-neutral by default |
| wi | we / us / our | "Wi fi fix dat" (We need to fix that) | |
| dem | they / them / their | "Dem file dem big" (Those files are big) | Also plural marker |
| unu | you all / you (plural) | "Unu ready?" (Are you all ready?) | |
| di | the | "Di function bruk" (The function is broken) | 1 token vs "the" |
| dat | that | "Dat a di issue" (That is the issue) | |
| dis | this | "Dis work good" (This works well) | |
| dah | that (demonstrative) | "Dah one deh" (That one there) | |
| weh | which / that / where | "Di file weh yuh edit" (The file that you edited) | Multi-purpose |
| smaddy | somebody / someone | "Smaddy change di config" (Someone changed the config) | |
| nutten | nothing | "Nutten nuh wrong" (Nothing is wrong) | |
| sumting | something | "Sumting off" (Something's off) | |
| everyting | everything | "Everyting irie" (Everything is fine) | |

## Verbs & Tense System

### Tense Markers

| Marker | Tense | Example |
|--------|-------|---------|
| (none) | Simple present / habitual | "Mi know" (I know) |
| did | Past | "Mi did fix it" (I fixed it) |
| a / de | Present continuous | "Mi a run di test" (I'm running the test) |
| ago / gwaan | Future | "Mi ago push it" (I will push it) |
| done | Completed | "Mi done already" (I've already finished) |
| ben | Past (alternate) | "Mi ben tell yuh" (I told you) |

| Patois | English | Example |
|--------|---------|---------|
| fi | to / for | "Mi haffi fi fix it" → "Mi haffi fix it" (I have to fix it) |

| ital | natural / organic / pure | "Keep di code ital" (Keep the code clean/natural) |
| livity | way of life / practice | "Dat a wi livity" (That's our practice) |
| irie | at peace / good / content | "Di system irie now" (The system is stable now) |
| Jah | God / the Most High | Cultural reference |
| reasoning | deep discussion / dialogue | "Mek wi have a reasoning bout dis" (Let's discuss this deeply) |
| forward | progress / advancement | "Wi a forward" (We're making progress) |
| heights | higher understanding / elevation | "Reach fi di heights" (Aim for excellence) |
